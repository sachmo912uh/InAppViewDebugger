HierarchyViewConfiguration

@objc(IAVDHierarchyViewConfiguration)
public final class HierarchyViewConfiguration : NSObject

Configuration options for the hierarchy (tree) view.

  • maxDepth

    The maximum depth that is rendered in the tree view. If the depth of the item exceeds this value, it will be hidden from view and can only be accessed by tapping the Subtree button of its parent node.

    Setting this to nil means there’s no maximum.
